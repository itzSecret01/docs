## 🇺🇸 How to Create and Connect Databases to FiveM in\
CreeperCloud

**Author:** itz_Secret\
**Discord:** `itz_Secret`\
**Hosting Provider:** [CreeperCloud](https://creepercloud.io/)

### 🛠️ Creating a MySQL Database on CreeperCloud

1. Go to https://creepercloud.io/ and log in to your dashboard.
2. Navigate to **Databases**.
3. Click **Create Database**.
4. Set:
   - **Name:** something like `fivem_db`
   - **Username** and **Password:** choose secure credentials
5. Once created, copy the following:
   - **Host/IP**
   - **Port**
   - **Username**
   - **Password**
   - **Database name**

### 🔌 Connecting the Database to Your FiveM Server

1. Open your **server.cfg** file.
2. Add the following line with your actual values:

   ```
   set mysql_connection_string "server=HOST;uid=USERNAME;password=PASSWORD;database=DBNAME"
   
   ```

   Example:

   ```
   set mysql_connection_string "server=127.0.0.1;uid=fivemuser;password=MyPass123;database=fivem_db"
   
   ```
3. Install a MySQL-compatible resource like **ghmattimysql** or **oxmysql**.

   For `oxmysql` (recommended):
   - Add it to your `resources` folder
   - Add `ensure oxmysql` in your `server.cfg`
4. Restart your server.

## 🇪🇸 Cómo Crear y Conectar Bases de Datos a FiveM usando CreeperCloud

**Autor:** itz_Secret\
**Discord:** `itz_Secret`\
**Proveedor de Hosting:** [CreeperCloud](https://creepercloud.io/)

### 🛠️ Crear una base de datos MySQL en CreeperCloud

1. Entra en https://creepercloud.io/ e inicia sesión en tu panel.
2. Ve a la sección **Databases** (Bases de Datos).
3. Haz clic en **Create Database** (Crear Base de Datos).
4. Configura:
   - **Nombre:** por ejemplo `fivem_db`
   - **Usuario y Contraseña:** crea credenciales seguras
5. Una vez creada, copia:
   - **Host/IP**
   - **Puerto**
   - **Usuario**
   - **Contraseña**
   - **Nombre de la base de datos**

### 🔌 Conectar la base de datos a tu servidor de FiveM

1. Abre tu archivo **server.cfg**.
2. Añade la siguiente línea con tus datos reales:

   ```
   set mysql_connection_string "server=HOST;uid=USUARIO;password=CONTRASEÑA;database=BASEDATOS"
   
   ```

   Ejemplo:

   ```
   set mysql_connection_string "server=127.0.0.1;uid=fivemuser;password=MiClave123;database=fivem_db"
   
   ```
3. Instala un recurso compatible con MySQL como **ghmattimysql** o **oxmysql**.

   Para `oxmysql` (recomendado):
   - Añádelo a tu carpeta `resources`
   - Pon `ensure oxmysql` en tu `server.cfg`
4. Reinicia tu servidor.
