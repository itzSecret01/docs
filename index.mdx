### ğŸ› ï¸ Creating a MySQL Database on CreeperCloud

1. Go to https://creepercloud.io/ and log in to your dashboard.
2. Navigate to **Databases**.
3. Click **Create Database**.
4. Set:
   - **Name:** something like `fivem_db`
   - **Username** and **Password:** choose secure credentials
5. Once created, copy the following:
   - **Host/IP**
   - **Port**
   - **Username**
   - **Password**
   - **Database name**

### ğŸ”Œ Connecting the Database to Your FiveM Server

1. Open your **server.cfg** file.
2. Add the following line with your actual values:

   ```
   set mysql_connection_string "server=HOST;uid=USERNAME;password=PASSWORD;database=DBNAME"
   
   ```

   Example:

   ```
   set mysql_connection_string "server=127.0.0.1;uid=fivemuser;password=MyPass123;database=fivem_db"
   
   ```
3. Install a MySQL-compatible resource like **ghmattimysql** or **oxmysql**.

   For `oxmysql` (recommended):
   - Add it to your `resources` folder
   - Add `ensure oxmysql` in your `server.cfg`
4. Restart your server.
