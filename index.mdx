## # Introduction to FiveM

## 🇺🇸 About This Project

Welcome to the documentation for the **FiveM servers configurations for CreeperCloud**\
created by **itz_Secret**.  

This documentation is designed to guide you through every step needed to properly configure and connect your FiveM server with external databases using **CreeperCloud**.

You’ll find complete instructions in both English and Spanish, including:

- How to create a MariaDB database for your FiveM server on CreeperCloud

- How to connect it to your FiveM server using `MariaDB` 

- Common issues and solutions

`Please use the menu on the left to access the language and topic of your choice.`
